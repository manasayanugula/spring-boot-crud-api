# Spring Boot CRUD API - User Management

This is a RESTful CRUD API built using Spring Boot.

## Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven
- Postman

## Features
- Create User
- Get All Users
- Get User By ID
- Update User
- Delete User

## API Endpoints

GET /users  
GET /users/{id}  
POST /users  
PUT /users/{id}  
DELETE /users/{id}  

## Postman Collection
A Postman collection is included in this repository:
`postman_collection.json`

You can import it into Postman to test all endpoints.
